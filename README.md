# Extintores

Aplicación web instalable para registrar revisiones de extintores y generar el Excel oficial.

## Publicación con GitHub Pages

1. Crear un repositorio público en GitHub llamado `extintores`.
2. Subir a la raíz del repositorio todos los archivos y la carpeta `vendor` incluidos aquí.
3. Abrir **Settings → Pages**.
4. En **Build and deployment**, seleccionar **Deploy from a branch**.
5. Seleccionar la rama **main**, la carpeta **/(root)** y pulsar **Save**.
6. Esperar unos minutos. La aplicación estará en `https://USUARIO.github.io/extintores/`.

No se necesita instalar dependencias ni ejecutar una compilación.
